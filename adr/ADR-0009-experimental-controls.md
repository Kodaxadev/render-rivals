# ADR-0009 — Random, Retain-Current, Oracle, and Linear Controls

**Status:** Accepted

## Context

Selector value cannot be isolated without controls using the same Candidate set and evidence conditions.

## Decision

Measure:

- evidence-backed selector;
- expected random eligible selection;
- always retain current;
- human-preferred eligible oracle;
- serious linear refinement using the same task brief.

The first pilot is exploratory rather than preregistered.

## Consequences

Selector uplift and opportunity rate become measurable. Analysis is more involved, and a small exploratory sample cannot certify production safety or general efficacy.
