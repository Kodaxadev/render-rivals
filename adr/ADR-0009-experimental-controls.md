# ADR-0009 — Random, Retain, Oracle, and Linear Controls

**Status:** Accepted

## Context

Selector value cannot be isolated without controls using same candidate set.

## Decision

Compute evidence selector, expected random eligible selection, retain champion, human oracle, and serious linear refinement. First pilot exploratory, not preregistered.

## Consequences

Selector value measurable; more analysis; small sample cannot certify safety.
